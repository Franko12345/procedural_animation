# Gameloop

Bullet Heaven / Survivor-like. Attack is **automatic**
([Weapons](./weapon.md)); the player only moves / dodges / positions.

## Loop

Themed [Round](./round.md) → enemies drip from nests → clear → **[Camp](./camp.md)**
(pollen shop + [Charms](./charm.md) + route) → next round.

XP raises the level → **3 cards** (new weapon / +weapon level / passive)
via [Evolution](./evolution.md). Eating or dash-killing carriers can
grant parts (rare).

## Downed / revive

Fallen = "down"; the coop partner revives by touching. All-down = end.

## Score / pollen

Score and pollen both scale with the **combo** (`game.combo`).
